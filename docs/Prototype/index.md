## Aanpak

Het prototype is gevalideerd via gebruikerstesten met leraren. Twee ontwerpers (Shehbaaz en Mazen) werkten onafhankelijk van elkaar een designrichting uit. De uiteindelijke [designverantwoording](designverantwoording.md) combineert de sterktes van beide ontwerpen.
